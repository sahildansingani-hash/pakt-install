Install Pakt

Add Pakt to your iPhone Home Screen for a full-screen, app-like experience.

Install Profile
⚠️ Configuration profiles only work on iPhone and iPad. Please open this page in Safari on your iOS device.

How to install

  1. Tap Install Profile above
  2. Tap Allow when prompted
  3. Open the Settings app
  4. Tap Profile Downloaded at the top
  5. Tap Install (top right) and enter your passcode
  6. Find the Pakt icon on your Home Screen

Must be opened in Safari on iOS. The profile is unsigned and safe — it only adds a Home Screen shortcut.
